# 25 February, 2025

These changes have been shipped with version `0.405` of the CLI.

## `File[]` and `Duration[]`

Support for arrays of type `File` and `Duration` are now supported in the schema and in functions.

## Reduced client bundle size

The file size of the client SDK generated by running `keel client` has been drastically reduced.
