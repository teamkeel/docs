# 29 September, 2023

These changes have been released with version `0.368` of the CLI.

## Summary

We are excited to announce the release of Keel `0.368`. This update ships a new feature to Keel.

## Events

With events you are able to asynchronously execute a subscriber function at the point when an _event_ happens in your application. 

Go to the [Events](/events) page in our documentation to get started.
