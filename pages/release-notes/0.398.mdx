# 16 January, 2024

## Improvements & bugfixes

This new release comes with a few smaller improvements and third party library updates:

* Removed legacy go-jose dependency
* Updated OpenID Connect client pacakges to the latest version (v3)
* Tools: Added support for tools to be displayed in a Grid layout
* GitHub workflows updates

---
Please make sure to update your CLI by running `npm install -g keel`.
