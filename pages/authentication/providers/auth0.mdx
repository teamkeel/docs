# Auth0

Not built-in, and so must be provided as a custom `oidc` provider.  For SSO Login, make sure to also specify a `redirectUrl`:
 For e.g.

```yaml
auth:
  tokens:
    accessTokenExpiry: 300
    refreshTokenExpiry: 6000

  redirectUrl: http://localhost:8000/signedin

  providers:
    - type: oidc
      name: Auth0
      issuerUrl: https://dev-1khdutlsd5lbqkvhv.us.auth0.com/
      clientId: 7NxsBWSA5qoBClGUdMskbcF0dQLKyY
```
