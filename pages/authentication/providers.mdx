# Providers

The table below lists the providers we support out of the box, however **you can configure any OpenID Connect provider you want** using the `oidc` provider type.

| Provider        | Type       | SSO Login | OIDC Token | 
|-----------------|------------|----------|------------|
| Google          | `google`   | ✅        | ✅          | 
| Facebook        | `facebook` | ✅        | ✅          |
| GitLab          | `gitlab`   | ✅        | ✅          |
| _Custom OIDC_   | `oidc`     | ✅        | ✅          | 

We have provided instructions on how to set up a client with each of the above. 

 - [Google](/authentication/providers/google) 
 - [Facebook](/authentication/providers/facebook) 
 - [GitLab](/authentication/providers/gitlab)
