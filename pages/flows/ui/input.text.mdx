# Text input

A text input field

```ts
ctx.ui.input.text("name", config);
```

## Config

import { PropDefinition } from "@components/propDefinition";

/>
The maximum length of the input.

<PropDefinition
  data={{

    type: "number",
  }}
/>
The minimum length of the input.

## Usage

Elements are rendered by by adding to a `page`. A page that contains input elements, will return an object with the keys matching the `name` property of each input.

```ts
const result = await ctx.ui.page({
  title: "Text inputs",
  content: [
    ctx.ui.input.text("name");
    ctx.ui.input.text("email", {
      label: "Email address",
      placeholder: "Enter your email address",
    }),
  ],
});

console.log(result.name);
console.log(result.email);
```
