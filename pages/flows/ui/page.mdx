# Page

A page is a layout element used to group multiple UI components, both inputs and displays, into a single logical unit. Pages represent

```ts
ctx.ui.page({ title, description, stage, content, actions, validate });
```

## Usage

```ts
await ctx.ui.page({
  stage: "setup",
  title: "Configure Workspace",
  description: "Set up your project preferences.",
  content: [
    ctx.ui.input.text("projectName", { label: "Project Name" }),
    ctx.ui.input.boolean("notifications", {
      label: "Enable Notifications",
      defaultValue: true,
    }),
  ],
  actions: [
    { label: "Continue", value: "continue", mode: "primary" },
    { label: "Cancel", value: "cancel" },
  ],
  validate: async (data) => {
    if (!data.projectName) return "Project name is required";
    return true;
  },
});
```
